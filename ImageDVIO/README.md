# ImageDVIO

## Description

> Votre ami a conçu son propre parseur d'image. Il souhaite le rendre populaire ! Cependant vous lui dites qu'il ne semble pas assez sécurisé à votre goût. 
> Prouvez lui les dangers de son programme en récupérant le secret présent dans /flag.

## Déploiement

```bash
cd deploy
docker compose up --build
```

http://127.0.0.1:8080/
